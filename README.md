# webpack-starter
Starter webpack project, including support for the most used frontend dependencies. See README.
# Project under active development ! Be aware of it might change often during its development.

## What's included ?
* Support for ES2015
* Support Sass/Scss
* Production minifications for css and js files

## Installation
With npm :
```
npm install
```
or with yarn :
```
yarn install
```

## Starting environments
Development :
```
npm run dev
```
or
```
yarn dev
```
Production build :
```
npm run build
```
or
```
yarn build
```
